# portfolio-cases — Yujie Hu

A curated set of case studies highlighting my work across full-stack web, Android, and cloud/serverless projects.  
Each case study summarises the problem, stack, architecture, my role & contributions, and links to original team repositories (when applicable).

## Cases

### 1) Grovvic — Sustainable Gardening Web App (React + Python, AWS/Vercel)
- **Role**: Frontend lead + UI/UX contributor; React setup & environment; Vercel deploy; core pages (plant recommendation & search + detailed guide, sustainability education + quiz, garden simulation); homepage & about; UI adjustments across pages.
- **Stack**: React, Python API, AWS, Vercel
- **Live**: https://netvivo-grovvic.vercel.app/
- **Official repo (team):** https://github.com/yujie-huu/Grovvic
- **Case page**: [./grovvic-case/README.md](./grovvic-case/README.md)

### 2) EcoFit — Climate-smart Wardrobe Android App (Kotlin + Firebase/Room/Retrofit)
- **Role**: Data layer owner (Room DB, entities, DAOs); UI + logic for Wardrobe screen (grid/list, search, add/delete items).
- **Stack**: Kotlin, Jetpack Compose, Firebase Auth/DB, Room, Retrofit, WorkManager
- **Original repo (team)**: https://github.com/gbjdyyds/fit5046-A4
- **Case page**: [./ecofit-case/README.md](./ecofit-case/README.md)

### 3) BirdTag — Serverless Bird Recognition (AWS)
- **Role**: AWS IAM; API Gateway; Cognito auth; `update_tags` Lambda (Python); SNS-based notifications by tag.
- **Stack**: AWS (S3, Lambda, API Gateway, DynamoDB, SNS, Cognito, CloudWatch), Python, (client: Vue, local only)
- **Original repo (team)**: https://github.com/dijia1124/cc-a3-group
- **Case page**: [./birdtag-case/README.md](./birdtag-case/README.md)
